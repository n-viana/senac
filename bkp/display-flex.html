Flex direction

A propriedade define que direção o contêiner que empilhar os itens flex

Seus valores são:

  • column: empilha os itens flexíveis verticalmente (de cima para baixo)
  • colum-reverse: empilha os itens flexíveis verticalmente (de baixo para cima)
  • row: empilha os itens flexíveis horizontalmente (da esquerda para direita)
  • row-reverse: empilha os itens flexíveis horizontalmente (da direita para esquerda)

Flex wrap

A propriedade especifica se os itens flexíveis devem ser ou não, de acordo com a viewport.

Seus valores são:

  • wrap: especifica que os itens flexíveis serão enrolados, se necessário
  • nowrap: especifica que os itens flexíveis não serão enrolados (modo padrão)
  • wrap-reverse: especifica que os itens flexíveis irão embrulhar se necessário, e, ordem inversa

Flex flow

A propriedade é uma propriedade abreviada para definir ambos o flex-direction e o flex-wrap. Para visualizar, basta usar o exemplo anterior inserindo o flex-flow: row wrap;, no lugar do flex-wrap.

Justify content

Propriedade é usada para alinhar os itens flexíveis na horizontalmente

Seus valores são:

  • center: Os valores de alinhamento flexíveis no centro do contêiner
  • flex-start:Os valores de alinhamento flexíveis no início do contêiner
  • flex-end: Os valores de alinhamento flexíveis no final do contêiner
  • space-around: Os valores de alinhamento flexíveis com espaço antes, entre e depois das linhas
  • space-between: Os valores de alinhamento flexíveis com espaço entre as linhas de linha

Align items

A propriedade é usada para alinhar os itens flexíveis na verticalmente

Seus valores são:

  • center: O valor alinha os itens flexíveis no meio do contêiner
  • flex-start: O valor alinha os itens flexíveis na parte superior do contêiner
  • flex-end: O valor alinha os itens flexíveis na parte inferior do contêiner
  • stretch: O valor estende os itens flexíveis para preencher o contêiner
  • baseline: O valor alinha os itens flexíveis, com as suas linhas base

Align content

A propriedade é usada para alinhar as linhas flexíveis

Seus valores são:

  • center: O valor mostra as linhas flexíveis no meio do contêiner
  • stretch: O valor estica as linhas flexíveis para assumir todo o espaço restante do contêiner
  • flex-start: O valor exibe as linhas flexíveis no início do contêiner
  • flex-end: O valor exibe as linhas flexíveis no final do contêiner
  • space-around: O valor exibe as linhas flexíveis com espaço antes, entre e depois dentro do contêiner
  • space-between: O valor exibe as linhas flexíveis com o mesmo espaço entre elas no contêiner

Neste exemplo, usamos um contêiner com 600px de altura, com o flex-wrap, propriedade definida para wrap, para melhor demostrar o align-content.
